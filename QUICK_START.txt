ğŸ¯ DDA ALGORITHM EXPLAINED
â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€

DDA (Digital Differential Analyzer) draws a line by:

1. Calculate dx = x2-x1, dy = y2-y1
2. Find steps = max(|dx|, |dy|)
3. Calculate increments = dx/steps, dy/steps
4. Loop: plot point, add increments
5. Round coordinates to get pixel positions

Example from (100,100) to (700,500):
   dx = 600, dy = 400
   steps = 600
   x_inc = 1.0, y_inc = 0.667
   Each step moves 1 pixel right, 0.667 pixels down
