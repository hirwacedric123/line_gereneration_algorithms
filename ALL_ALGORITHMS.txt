DDA Line:
   1. Calculate dx = x2-x1, dy = y2-y1
   2. Find steps = max(|dx|, |dy|)
   3. Calculate x_inc = dx/steps, y_inc = dy/steps
   4. Loop: plot(round(x), round(y)), x += x_inc, y += y_inc

Bresenham Line:
   1. Calculate dx = |x2-x1|, dy = |y2-y1|
   2. Determine step directions sx, sy (Â±1)
   3. Initialize error = dx - dy
   4. Loop: plot(x,y), adjust x or y based on 2*error

Bresenham Circle:
   1. Start at (0, radius)
   2. Initialize decision d = 3 - 2*radius
   3. Loop while x â‰¤ y:
      - Plot 8 symmetrical points
      - Increment x
      - Update d and possibly decrement y
